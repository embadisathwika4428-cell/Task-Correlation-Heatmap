# Task-Correlation-Heatmap
This project demonstrates correlation analysis using Python, Pandas, Seaborn, and Matplotlib. A correlation matrix was generated to identify relationships between variables, and the results were visualized using a heatmap. The project helps in understanding data patterns, feature relationships, and exploratory data analysis techniques.
